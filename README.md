# Basic WebSocket

A basic WebSocket(Client, Server)

## Technologies

- Node.js
- Docker
- FastAPI
- WebSocket
- Python

## Setup and run

```code
# Clone the repo
git clone https://github.com/hesbon-osoro/FastAPI-Node-Docker.git

# go to ...
cd FastAPI-Node-Docker

# Docker compose build
docker-compose up --build

# Open the app
http://0.0.0.0:8080/
```
